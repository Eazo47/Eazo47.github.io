更换主题

hexo的所有主题都在下面这个地址：

https://hexo.io/themes/

如何下载主题

方法1

打开主题的项目地址后，点击Clone or download

然后复制https开头的地址

然后在你的博客文件夹下面打开git bash

执行：

git clone 代表克隆，后面接上一串下载地址，最后面的themes/hexo是把文件夹下载到主题目录下，并把主题文件命名为hrxo

方法2

直接在github中将主题文件打包下载下来

然后解压压缩包，把文件夹放在博客目录下的themes目录，并把文件夹命名为主题的名字

修改配置文件

在博客的目录下有一个叫_config.yml的文件

将它打开找到themes这一行，然后将主题名字修改为next，主题便成功更换了
